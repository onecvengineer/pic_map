# Pic Map v1 Product

## 目标

Pic Map 是私有照片地图 + 旅行时间轴系统。v1 面向单用户自部署：通过网页批量上传照片，服务端入库、去重、解析 EXIF/GPS、生成预览，然后在地图和时间轴中浏览与修正位置。

## 双入口

### 地图工作台

- OSM/Leaflet 展示 WGS84 坐标。
- 桌面端采用深色筛选侧栏、地图主画布、右侧详情和底部照片胶片条的工作台布局。
- 支持按文件名、设备、日期、地点、GPS 状态搜索。
- 支持查看待确认照片。
- 支持高德地点搜索；高德 GCJ-02 坐标进入系统前转为 WGS84。
- 支持单张手动赋位、确认推断位置、清除位置。

### 旅行时间轴

- 按拍摄日期分组。
- 顶部展示路线横幅，突出有 GPS 的旅行停靠点。
- 每天展示照片流、设备数量、有位置数量和待确认数量。
- 点击有位置照片会跳转回地图点位。
- 缺少 GPS、自动推断和待确认状态会显示在照片卡片与详情里。

## 移动端

v1 是响应式 Web。桌面端地图工作台使用左侧工具栏、中间主视图、右侧详情；旅行时间轴是独立全宽旅程视图，不复用地图工作台侧栏。移动端使用底部 Tab：地图、时间线、上传、搜索、详情。

## 导入流程

1. 用户在网页批量选择照片或文件夹。
2. 服务端保存原图到 `data/originals/`。
3. 服务端计算 SHA-256，重复原图只保留一份。
4. 服务端生成浏览器可显示预览到 `data/previews/`。
5. 服务端解析 EXIF/GPS 和拍摄参数。
6. 无 GPS 照片按默认前后 30 分钟寻找最近有 GPS 照片，生成 `inferred + pending` 位置。

## v1 范围

- 做单用户密码登录。
- 做网页批量上传，不依赖扫描本机目录。
- 做地点搜索缓存、位置推断和手动修正。
- 做迁移 manifest 和数据库快照导出。
- 暂不做多用户、公开分享、人物识别、复杂相册和 App/小程序。
